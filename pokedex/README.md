# Pokedex

TypeScript based CLI application for interacting with the Pokemon web API. Built as
part of the
[Build a Pokedex in TypeScript](https://www.boot.dev/courses/build-pokedex-cli-typescript)
Boot.dev project.

## Usage

Run the TypeScript compiler:

```sh
$ npm run build
```

Run the app:

```sh
$ npm run start
```

Run tests:

```sh
$ npm run test
```
