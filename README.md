# BootDotDev

Collection of Python based projects completed as part of the
[Boot.dev](https://boot.dev) training platform.

## Project List
1. [WorldBanc](https://github.com/drewlee/bootdotdev/tree/main/worldbanc) - For exercises in using the Unix terminal.
2. [BookBot](https://github.com/drewlee/bootdotdev/tree/main/python-bookbot) - CLI application for text file analysis.
3. [Asteroids Game](https://github.com/drewlee/bootdotdev/tree/main/asteroids) - Simple clone of the Asteroids game.
4. [AI Agent](https://github.com/drewlee/bootdotdev/tree/main/ai-agent) - LLM-based CLI application which uses the Google Gemini API.
5. [Static Site Generator](https://github.com/drewlee/bootdotdev/tree/main/static-site-gen) -
Static site generator. Transforms markdown documents to HTML pages.
