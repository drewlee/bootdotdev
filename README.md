# BootDotDev

Collection of projects completed as part of the [Boot.dev](https://boot.dev) learning
platform.

## Project List
1. [WorldBanc](https://github.com/drewlee/bootdotdev/tree/main/worldbanc) - For
exercises in using the Unix terminal.
2. [BookBot](https://github.com/drewlee/bootdotdev/tree/main/python-bookbot) - Python
CLI application for text file analysis.
3. [Asteroids Game](https://github.com/drewlee/bootdotdev/tree/main/asteroids) - Simple
Python based clone of the Asteroids game.
4. [AI Agent](https://github.com/drewlee/bootdotdev/tree/main/ai-agent) - LLM-based
Python CLI application using the Google Gemini API.
5. [Static Site Generator](https://github.com/drewlee/bootdotdev/tree/main/static-site-gen) -
Static site generator. Transforms markdown documents to HTML pages.
