# Static Site Generator

Transforms markdown documents to HTML pages. Built as part
of the
[Build a Static Site Generator in Python](https://www.boot.dev/courses/build-static-site-generator-python)
Boot.dev course.

Generated site deployed to
https://drewlee.github.io/bootdotdev/static-site-gen/.

## Usage

Generating pages:

```sh
$ ./main.sh
```

Running tests:

```sh
$ ./test.sh
```
